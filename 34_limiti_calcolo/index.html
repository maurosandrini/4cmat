Il problema: \(\frac{\infty}{\infty}\) ?

Consideriamo questa funzione razionale:

\[ f(x) = \frac{5x^2 - 3x + 1}{2x^2 + 7} \]

Cosa succede quando \(x \to +\infty\)?

Il numeratore \(\to \infty\) e il denominatore \(\to \infty\).

\(\dfrac{\infty}{\infty} = \;?\)

NON si puo' semplificare! E' una forma indeterminata.

Serve un metodo per risolvere. Vediamo due strade.

La regola: confronta i gradi

Per \(\lim_{x\to\pm\infty} \dfrac{P(x)}{Q(x)}\) con \(P\) di grado \(n\) e \(Q\) di grado \(m\):

CASO 1: \(n < m\)

Il den domina

\[\text{Limite} = 0\]

La frazione si schiaccia verso zero.

AO: \(y = 0\)

CASO 2: \(n = m\)

Pari grado

\[\text{Limite} = \frac{a}{b}\]

Rapporto dei coefficienti del grado massimo.

AO: \(y = \frac{a}{b}\)

CASO 3: \(n > m\)

Il num domina

\[\text{Limite} = \pm\infty\]

La funzione esplode.

AO: non esiste!

CASO 1   Grado num < grado den → 0

\[ \lim_{x\to\pm\infty} \frac{ax^n + \ldots}{bx^m + \ldots} = 0 \quad \text{se } n < m \]

Esempio:

\[ \lim_{x\to+\infty} \frac{3x + 1}{x^2 - 4} \]

Grado num = 1, grado den = 2

Poiche' \(1 < 2\): il denominatore domina

\(\Rightarrow\) Limite = \(\boxed{0}\)   AO: \(y = 0\)

CASO 2   Grado num = grado den → \(\frac{a}{b}\)

\[ \lim_{x\to\pm\infty} \frac{ax^n + \ldots}{bx^n + \ldots} = \frac{a}{b} \]

Esempio:

\[ \lim_{x\to-\infty} \frac{7x + 2}{x - 3} \]

Grado num = 1, grado den = 1

Gradi uguali! Coefficienti: \(\frac{7}{1}\)

\(\Rightarrow\) Limite = \(\boxed{7}\)   AO: \(y = 7\)

Attenzione: guarda SOLO i coefficienti del termine di grado massimo. Ignora tutto il resto!

CASO 3   Grado num > grado den → \(\pm\infty\)

\[ \lim_{x\to\pm\infty} \frac{ax^n + \ldots}{bx^m + \ldots} = \pm\infty \quad \text{se } n > m \]

Esempio:

\[ \lim_{x\to+\infty} \frac{2x^3 - x}{x^2 + 1} \]

Grado num = 3, grado den = 2

Poiche' \(3 > 2\): il numeratore domina

Semplifica: \(\frac{2x^3}{x^2} = 2x \to +\infty\)

\(\Rightarrow\) Limite = \(\boxed{+\infty}\)   Nessun AO!

Per \(x \to -\infty\): \(2x \to -\infty\), quindi limite = \(\boxed{-\infty}\)

Errori da evitare

Errore 1

\[\frac{\infty}{\infty} = 1\]

SBAGLIATO!

\(\frac{\infty}{\infty}\) e' una forma indeterminata. Il risultato dipende da COME numeratore e denominatore vanno a infinito.

Usa la regola dei gradi!

Errore 2

Dimenticare il segno nel Caso 3

\[\lim_{x\to-\infty} \frac{2x^3}{x^2} = 2x\]

A \(+\infty\): \(2x \to +\infty\)

A \(-\infty\): \(2x \to \mathbf{-\infty}\)

Sostituisci mentalmente \(x\) con un numero negativo grande!

Errore 3

\[\frac{6x^2 + x}{3x^2 - 5}\]

Grado den = 2, non 3!

Gradi uguali → \(\frac{6}{3} = 2\)
